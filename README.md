# AI-Powered LED Advertising Booking System

This project is an AI-driven LED advertisement booking system that automates
availability checking, price estimation, booking confirmation, updates, and cancellations.

## Features
- Conversational AI assistant (Vapi)
- Real-time availability via Google Calendar
- Price estimation before booking
- Booking, update, and cancellation workflows
- Supabase-based booking storage
- Telegram notifications

## Tech Stack
- Vapi AI
- n8n
- Google Calendar API
- Supabase
- Telegram Bot

## How It Works
1. User interacts with AI assistant
2. System checks availability and gives estimate
3. Booking is confirmed only after user approval
4. Calendar and database are updated automatically
